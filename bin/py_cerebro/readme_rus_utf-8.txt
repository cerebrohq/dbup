************************************
py_cerebro - програмный интерфейс на языке Python для доступа к базе данных и файловому хранилищю Cargador
************************************

Использует библиотеку Python версии 3.x

py_cerebro имеет 4 модуля:

database
dbtypes
cargador
cclib


* Модуль database
-------------------
Содержит функционал доступа к базе данных

* Модуль dbtypes
-------------------
Содержит описания полей и флагов базы данных

* Модуль cargador
-------------------
Содержит функционал доступа к файловому хранилищу Cargador

* Модуль cclib
-------------------
Содержит общие полезные функции
